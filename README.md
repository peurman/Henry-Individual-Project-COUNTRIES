## About the Web

Hi everyone! This website was created as an Individual Project for the certification of Full Stack Developer offered by **`Henry's Bootcamp`**.

The theme was about **`Countries`** of the world, following certain mandatory guidelines:

‚óΩ Get the information from an external API, and create an own database to fill it with that information. <br/>
‚óΩ Create tourist activities through a controlled form, which verifies data before allowing the upload, and whose information also was stored in the database.<br/>
‚óΩ Include a Landing Page.<br/>
‚óΩ Display countries with flag, name and continent.<br/>
‚óΩ To be able to sort them alphabetically or by number of inhabitants.<br/>
‚óΩ To be able to search for countries that "contain" the text searched, filter them by Continent and by Activities.<br/>
‚óΩ Show the detail of each country (capital, surface, population, etc).<br/>
‚óΩ Paginated with 9 countries on page 1, and then 10 countries per page.<br/>
‚óΩ Use Vanilla CSS.<br/><br/>
I added to this:<br/>
‚óΩ Combination of the three filters and sorting.<br/>
‚óΩ Next and previous buttons.<br/>
‚óΩ 9 possible pop-up messages (to confirm, inform or warn).<br/>
‚óΩ 1 "Loading..." message while loading information.<br/>
